# 📚 欢迎来到你的知识库

这是你的个人知识库，你可以在这里存放和整理各种文件和笔记。

## 📁 目录说明

| 目录 | 用途 |
|------|------|
| **技术** | 编程、开发、技术文档 |
| **学习笔记** | 课程笔记、读书摘要 |
| **工作文档** | 工作相关的文件和资料 |
| **参考资料** | 收藏的文章、教程、手册 |

## 🚀 快速开始

1. 点击左侧边栏浏览文件目录
2. 使用顶部搜索框搜索内容
3. 点击 **新建笔记** 创建 Markdown 笔记
4. 直接将文件拖入对应目录即可归档

## ✏️ Markdown 语法提示

- `# 标题` — 标题
- `**粗体**` — **粗体**
- `- 列表` — 列表
- `` `代码` `` — `代码`
- `[链接](url)` — 链接

祝你使用愉快！🎉
